#The game features a full world map system with multiple explorable wasteland areas, including farms, supply outposts, industrial ruins, forbidden lands, military bases, and ancient altars. Players can switch between areas on the world map and enter each zone to explore combat maps. It includes a complete weapon and projectile combat system: multiple unlockable weapons such as SMG, machine gun, laser sword and piercing laser gun. Each weapon has independent fire rate, damage, range and projectile speed. Projectiles support collision detection with walls, map boundaries and enemies, plus piercing mechanics and lifetime fading. The game has enemy AI spawning, health deduction, death loot dropping, and different physics modes for platformer levels and top-down levels. It also provides custom world map UI rendering, random background crack/dust effects, area connection paths, and unique icon drawing for each region.
#Complementing the survival gameplay is a rich farm simulation RPG system, where players can tend to crops and manage resources to build a thriving homestead. The farm mode includes day-night cycles, crop growth stages with water/fertilizer mechanics, and seasonal harvest cycles, all optimized for embedded MCUs. Players can craft tools and upgrade equipment at a dedicated forge, trade goods with NPCs in the outpost, and engage in casual fishing minigames with motion-controlled catch mechanics.
#All logic is lightweight and optimized for low-resource embedded MCUs, with modular code for map management, weapon firing, projectile updating, enemy logic, world map interaction, and farm simulation loops.
